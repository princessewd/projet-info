# Description :
Ce projet est un jeu inspiré de Candy Crush. Le jeu propose une grille de bonbons où le joueur peut échanger des bonbons adjacents pour former des combinaisons. 
Chaque combinaison rapporte des points, et des bonus spéciaux apparaissent pour des alignements particuliers.

# Langages utilisés :
C uniquement

# Fonctionnalités :
Grille interactive avec bonbons générés aléatoirement
Détection automatique des combinaisons horizontales et verticales
Calcul,affichage et sauvegarde du score
Bonus pour des combinaisons spéciales (ligne complète, explosion, etc.)
Niveaux à thèmes
Interface utilisateur simple avec écran de fin et possibilité de recommencer

# Installation et utilisation :
- Cloner le projet depuis le dépôt Git
- Installer les dépendances backend si nécessaire
- Lancer le serveur backend 
- Ouvrir le frontend dans le navigateur (index.html)
- Jouer en cliquant ou en glissant les bonbons adjacents pour créer des combinaisons
- Les combinaisons de 3 bonbons ou plus disparaissent et rapportent des points
- Les bonus apparaissent pour des combinaisons spéciales
- Le score est mis à jour en temps réel
- La partie se termine après un nombre limité de mouvements ou lorsque la grille n’a plus de mouvements possibles
